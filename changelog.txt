- Version 0.3.1
    - Additions
        - Added corruption mob effect which locks off some hearts for the duration, but also provides strength IV for the duration
        - Add Totem of Last Breath which revives the player with five hearts, and applies corruption
    - Changes, Tweaks & Updates
        - Living Sculk will now convert some items in the player's inventory to their corrupted variants (i.e. totem of undying)
            - Note that this will never affect any 'irreplaceable' items such as armor, tools, weapons, enchanted books, etc.
    - Bug Fixes

- Version 0.3.0
    - Added new biome: Sculk Shore, in order to ease the transition between beaches and other biomes
    - Added new gravel variant: Sculk Gravel; generates in Shroud similar to the overworld with similar properties
    - Added new variant of the Sculk tree: Scraggle tree, for use in transitory biomes
    - Added Gloomstone and Gloomstone dust, a corrupted glowstone alternative
    - Added Ghost Bloom, a new inverted plant which grows on the underside of leaves
    - Adjusted sculk arch feature to vary the size more
    - Removed sculk emitter from Sculk Beach biome
    - Reduced frequency of sculk patch spawns; it was overwhelming other biome generation
    - Fixed issue where the sculk emitter would always emit particles upwards instead of where the 'emitter' texture was facing
    - Fixed references in lake_lava placed features to reference the custom lake_lava configured feature
    - Fixed translation for sculk_stone and sculk_deepslate versions of Eventide ore
